Formula Interpretation
Catalan number Cn = (2n)! / ((n + 1)! n!)

Triangular number n(n+1)/2

These are used to count the number of ways to divide a polygon into triangles.
Harmonic number Hn = Hn−1 + 1/n

These are the numbers that can be represented as the sum of the integers from 1 to n.
Fibonacci number Xn = Xn−1 + Xn−2

These are the sum of the reciprocals of the positive integers. Fibonacci numbers: These are a sequence of numbers where each number is the sum of the two preceding ones.
Lucas number Ln = Ln-1 + Ln-2

Eulerian number An = (n-1)! ∑ (k+1)C(n, k)

These are a related sequence of numbers that start with 2 and 1 instead of 0 and 1.
Stirling number S(n,m) = m! ∑ (-1)^k C(m, k) ∑ k^(n-k)

The number of ways of partitioning a set of n elements into m nonempty sets.
