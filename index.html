Formula Interpretation
Catalan number C n = ( 2 n ) ! ( n + 1 ) ! n !

Catalan numbers are defined as a mathematical sequence that consists of positive integers, which can be used to find the number of possibilities of various combinations.
Triangular number T n = n ( n + 1 ) 2

The nth triangular number is the number of dots in the triangular arrangement with n dots on each side, and is equal to the sum of the n natural numbers from 1 to n.
Harmonic number H n = ∑ k = 1 n 1 k

In mathematics, the n-th harmonic number is the sum of the reciprocals of the first n natural numbers.
Fibonacci number F 0 = 0 , F 1 = 1 F n = F n - 1 + F n - 2

In mathematics, the Fibonacci sequence, in which each number is equal to the sum of the preceding two numbers.
Lucas number L 0 = 2 , L 1 = 1 L n = L n - 1 + L n - 2

Lucas numbers are similar to Fibonacci numbers. Lucas numbers are also defined as the sum of its two immediately previous terms. But here the first two terms are 2 and 1 whereas in Fibonacci numbers the first two terms are 0 and 1 respectively.

Eulerian number E ( m n ) = n E ( m - 1 n ) + m E ( m n - 1 )

In combinatorics, the Eulerian number A(n, m) is the number of permutations of the numbers 1 to n in which exactly m elements are greater than the previous element (permutations with m "ascents").
Stirling number S ( n m ) = 1 n ! ∑ k = 0 n ( - 1 ) k ( n n - k ) ( n - k ) m

The number of ways of partitioning a set of n elements into m nonempty sets.
