1. 首頁為index.html。
2. 點擊"show"，可以看到關於該公式的介紹，再按一下可收回。
3. "computet"為計算對應公式的結果；"Computet All"為計算該table的所有結果。
4. part 2 (Catalan number, Triangular number, Harmonic number, Fibonacci number, Lucas number)，中 input n 所計算出的是「列出第 0 ~ n 項結果」。
5. part 3 (Eulerian number, Stirling number) 則為「列出前 n 項結果」，會這樣是因為上網看，發現普遍在列出結果的時候(e.g. Euler’s Number Triangle, Stirling's Triangles)，幾乎沒有把第 0 項放上去。
