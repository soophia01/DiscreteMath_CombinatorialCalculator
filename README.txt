1. 首頁為index.html。
2. 點擊"show"，可以看到關於該公式的介紹，再按一下可收回。
3. "Compute"為計算對應公式的結果；"Compute All"為計算該table的所有結果。
4. part 2 (Catalan number, Triangular number, Harmonic number, Fibonacci number, Lucas number)，中 input n 所計算出的是「列出第 0 ~ n 項結果」。
5. part 3 (Eulerian number, Stirling number) 則為「列出第 n 列的所有結果」。
6. 關於 4. 和 5. 有特別在"Compute"寫提示文字告知使用者 (滑鼠碰觸時顯示)
