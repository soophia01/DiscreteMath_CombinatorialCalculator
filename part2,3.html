Formula Interpretation
Catalan number Cn = (2n)! / ((n + 1)! n!)

Triangular number n(n+1)/2

Harmonic number Hn = Hn−1 + 1/n

Fibonacci number Xn = Xn−1 + Xn−2

Lucas number Ln = Ln-1 + Ln-2

Eulerian number An = (n-1)! ∑ (k+1)C(n, k)

Stirling number S(n,m) = m! ∑ (-1)^k C(m, k) ∑ k^(n-k)

The number of ways of partitioning a set of n elements into m nonempty sets
